hexo+github博客搭建笔记

  • 安装Node.jd

    点击此处访问官网，下载相应的版本

    下载完成后进行安装，选项默认，一直点击next即可
    安装完成后,同时按win键+R键，输入cmd打开页面，再输入

    $ git config –global user.name “你的用户名”
    $ git config –global user.email “你的邮箱”

    这时你就可以通过https://你的用户名.github.io访问你的博客了
